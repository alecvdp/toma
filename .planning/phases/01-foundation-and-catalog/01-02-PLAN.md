<task type="auto">
  <name>Task 1: Create app entrypoint and catalog page with full CRUD UI</name>
  <files>app.py, pages/catalog.py</files>
  <action>
1. Create app.py:
   - st.set_page_config(page_title="Toma", page_icon=":pill:", layout="wide")
   - from db import init_db; init_db()
   - catalog_page = st.Page("pages/catalog.py", title="Catalog", icon=":material/medication:")
   - pg = st.navigation([catalog_page]); pg.run()

2. Create pages/catalog.py with these sections in order:

   a. SEARCH AND FILTER BAR (top of page):
      - st.text_input for name search (key="search_query")
      - st.selectbox for category filter with "All" option, populated from item_service.get_categories()
      - Call search_items(query, category) to get filtered results

   b. ADD ITEM FORM (collapsible via st.expander):
      - Wrap in st.expander("Add New Item")
      - Inside: st.form("add_item_form", clear_on_submit=True)
      - Fields: name (text_input, required), category (selectbox: supplement/prescription/vitamin/other), default_dosage (number_input, min=0.0, step=0.1), dosage_unit (selectbox: mg/mcg/IU/mL/capsule/tablet), notes (text_area)
      - On submit: validate name not empty, call create_item(), st.success(), st.rerun()

   c. EDIT ITEM FORM (shown when st.session_state.editing_item is set):
      - Load item via get_item(st.session_state.editing_item)
      - st.form("edit_item_form") pre-populated with current values
      - Same fields as add form
      - Save button: call update_item() with changed fields, del editing_item, st.rerun()
      - Cancel button: del editing_item, st.rerun()

   d. DELETE CONFIRMATION (shown when st.session_state.deleting_item is set):
      - Use st.warning() to show "Delete {item_name}?"
      - Two columns: Confirm (calls deactivate_item, dels state, reruns) and Cancel (dels state, reruns)

   e. CARD VIEW (main content):
      - Iterate over filtered items
      - Each item in st.container(border=True):
        - 3 columns: [3, 1, 1] ratio
        - Col 1: st.subheader(name), st.caption(description) if exists, st.text(notes) if exists
        - Col 2: st.metric("Dosage", f"{dosage} {unit}") if dosage exists, st.badge(category)
        - Col 3: Edit button (sets session_state.editing_item), Delete button (sets session_state.deleting_item)
      - Show st.info("No items found") if list is empty

   Pages MUST import from services.item_service only (never from db directly).
   Use st.form for all input to avoid rerun-on-keystroke issues (Pitfall 1 and 4 from research).
